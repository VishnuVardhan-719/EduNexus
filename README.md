# EduNexus 🎓

EduNexus is an AI-assisted student productivity and academic management platform designed to help students organize their studies, track academic performance, and improve focus.

The system integrates study planning, CGPA tracking, Pomodoro focus sessions, and performance analytics into a single intelligent dashboard.

---

## 🚀 Features

- 📊 **Student Dashboard** – Overview of academic performance and productivity
- 🧮 **CGPA Calculator** – Calculate and monitor cumulative grade point average
- 📅 **Study Planner** – Manage study tasks and deadlines
- ⏱ **Pomodoro Focus Engine** – Structured focus sessions for better concentration
- 📈 **Performance Analytics** – Visual insights into academic performance
- 🤖 **Smart Study Assistant** – Intelligent study recommendations
- 🔔 **Smart Notifications** – Alerts for tasks and deadlines

---

## 🏗 System Architecture

EduNexus follows a **Layered MVC Architecture**:
Client (Browser)
↓
Frontend (HTML, CSS, JavaScript, Bootstrap)
↓
Spring Boot Controllers
↓
Service Layer (Business Logic)
↓
Repository Layer
↓
SQLite Database
